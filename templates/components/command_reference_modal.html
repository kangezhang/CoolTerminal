命令参考手册

文件和目录

dir 列出目录内容
cd [路径] 切换目录
mkdir [目录名] 创建目录
rmdir [目录名] 删除目录
copy [源] [目标] 复制文件
move [源] [目标] 移动文件
del [文件] 删除文件
type [文件] 显示文件内容

系统信息

systeminfo 显示系统信息
hostname 显示计算机名
ver 显示系统版本
whoami 显示当前用户

网络

ipconfig 显示IP配置
ping [地址] 测试网络连接
netstat 显示网络连接
tracert [地址] 追踪路由

进程和服务

tasklist 列出所有进程
taskkill /PID [进程ID] 终止进程
sc query 查询服务状态

文件和目录

ls [-la] 列出目录内容
cd [路径] 切换目录
pwd 显示当前目录
mkdir [目录名] 创建目录
rm [-rf] [文件/目录] 删除文件或目录
cp [-r] [源] [目标] 复制文件或目录
mv [源] [目标] 移动或重命名
cat [文件] 显示文件内容
find [路径] -name [名称] 查找文件

系统信息

uname -a 显示系统信息
sw_vers 显示 macOS 版本
hostname 显示主机名
whoami 显示当前用户
df -h 显示磁盘使用情况
top 显示进程和资源

网络

ifconfig 显示网络配置
ping [地址] 测试网络连接
netstat -an 显示网络连接
curl [URL] 传输数据

进程和权限

ps aux 列出所有进程
kill [PID] 终止进程
sudo [命令] 以管理员权限执行
chmod [权限] [文件] 修改文件权限
chown [用户] [文件] 修改文件所有者

Homebrew (包管理)

brew install [包名] 安装软件包
brew update 更新 Homebrew
brew list 列出已安装的包

文件和目录

pwd 显示当前目录
mkdir [-p] [目录名] 创建目录

cp [-r] [源] [目标] 复制文件或目录
mv [源] [目标] 移动或重命名
cat [文件] 显示文件内容
grep [模式] [文件] 搜索文本
find [路径] -name [名称] 查找文件

系统信息

uname -a 显示系统信息
lsb_release -a 显示发行版信息
hostname 显示主机名
whoami 显示当前用户
df -h 显示磁盘使用情况
free -h 显示内存使用情况
top / htop 显示进程和资源
uptime 显示系统运行时间

网络

ip addr 显示网络接口
ping [地址] 测试网络连接
netstat -tulpn 显示网络连接
ss -tulpn 显示套接字统计
curl [URL] 传输数据
wget [URL] 下载文件

进程和权限

ps aux 列出所有进程
kill [-9] [PID] 终止进程
sudo [命令] 以 root 权限执行
chmod [权限] [文件] 修改文件权限
chown [用户]:[组] [文件] 修改文件所有者

包管理

apt update 更新包列表 (Debian/Ubuntu)
apt install [包名] 安装软件包 (Debian/Ubuntu)
yum install [包名] 安装软件包 (RedHat/CentOS)
dnf install [包名] 安装软件包 (Fedora)

系统管理

systemctl status [服务] 查看服务状态
systemctl start [服务] 启动服务
systemctl stop [服务] 停止服务
systemctl enable [服务] 设置服务开机自启
